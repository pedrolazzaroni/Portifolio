LazzHub

Este foi um projeto de uma plataforma que permite aos usuários gerar resumos inteligentes, criar questões de prova personalizadas e obter respostas sobre diversos assuntos, tudo com o auxílio de inteligência artificial Gemini. Desenvolvido com Laravel e estilizado com Tailwind CSS, o LazzHub é totalmente responsivo e está disponível para acesso em pedrolazzaroni.com.br/lazzhub. O código-fonte pode ser encontrado no meu GitHub.
